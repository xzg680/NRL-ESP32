NRL ESP32 USB Firmware Upgrade

Upgrade the device firmware from Chrome or Edge with a USB cable.

Language
Recommended: automatic upgrade

Use this when you just want to install the official firmware provided by this page. Connect the device with USB, click the button, choose the serial port, and follow the browser prompts.

Board model
This browser does not support Web Serial. Use Chrome or Edge.

Serial monitor

View device serial output directly. The browser will reconnect to the same USB port automatically after the device reboots.


    


    
How to upgrade

    
    
      
  1. Use Chrome or Edge on a computer.
    2. 
      
  2. Connect the device to the computer with a USB data cable.
    3. 
      
  3. Click Install firmware and choose the ESP32-S3 USB serial port.
    4. 
      
  4. Keep the device connected until the page says the upgrade is complete.
    5. 
      
  5. If the device is not detected, hold BOOT while connecting USB, then try again.
    6.
