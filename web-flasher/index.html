NRL ESP32 USB Web Flasher

Flash the ESP32-S3 radio bridge firmware from a browser over USB serial.

Firmware target: ESP32-S3 app0, version 0.0.1

Use Chrome or Edge on a desktop system. Connect the board through USB, then start installation.

This browser does not support Web Serial. Use Chrome or Edge.

Before Flashing

  1. Build and stage the firmware files with platformio run -e app0_main, then python scripts/stage_web_flasher.py.
  2. Serve this directory over HTTPS or from localhost. For local testing, run python -m http.server 8000 -d web-flasher.
  3. Open http://localhost:8000 in Chrome or Edge.
  4. Hold BOOT if your board does not enter download mode automatically.

Flash Layout

  • bootloader.bin at 0x0000
  • partitions.bin at 0x8000
  • boot_app0.bin at 0xE000
  • firmware.bin at 0x010000

Notes

The firmware path follows the project's dual OTA layout from part.csv. USB flashing installs the partition table needed for later WiFi web updates.
